What's in?

JavaCard is an amazing platform. Unfortunately, it lacks certain essential features which makes development troublesome and limiting. The OpencryptoJC project releases tools that aim to aid both experienced and new developers:JCMathLib implements
Integers, Big Numbers and Elliptic Curves for all, while the JCProfiler is the first free profiler for JavaCards.
